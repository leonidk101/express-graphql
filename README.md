# Pre-requirements 

## Node.js version

Make sure you're running the correct version of Node.js (requires [nvm](https://github.com/nvm-sh/nvm))

```bash
nvm i && nvm use
```

## Install yarn

### Enable corepack

```bash
corepack enable
```

# Install Dependencies

```bash
yarn install
```

# Development

```bash
yarn dev 
```
