# Pre-requirements 

### Make sure you're running the correct version of Node.js (requires [nvm](https://github.com/nvm-sh/nvm))

```bash
nvm i && nvm use
```

### Enable corepack

```bash
corepack enable
```

# Development 

### Install dependencies

```bash
yarn install
```

### Run express server

```bash
yarn dev 
```
